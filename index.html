Use our simple recipe to make data discoverable and reusable

  • Repositories

    Organize your data in topical repositories, making it findable and accessible. Explore and reuse open data provided by the community.

  • Datasets

    Define datasets in your data files. Use interactive digitalization tool to extract datasets from graphs, or simply load your data from text files or spreadsheets.

  • Relations

    Use embeded flexible curve-fitting tools to define functional relations in datasets. Apply smooth spline functions or fine-tune spline curves to achieve best fit tolerance.

  • API

    Reuse constructed curves in your models and calculations by pulling them directly into your code over SplineCloud API or by using our open-source client libraries.
